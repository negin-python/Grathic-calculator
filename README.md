# Grathic Calculator
ماشین‌حساب گرافیکی ساده با استفاده از Tkinter برای تمرین GUI در پایتون

## قابلیت‌ها
- جمع، تفریق، ضرب، تقسیم
- طراحی ساده و کاربرپسند با Tkinter
- قابل اجرا روی ویندوز
